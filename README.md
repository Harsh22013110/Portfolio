# Harsh Shukla — Portfolio

Industry-standard portfolio site built with **React** and **Vite**. Showcases experience, projects, skills, education, and contact information.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build for production

```bash
npm run build
```

Output is in `dist/`. Deploy to Vercel, Netlify, or GitHub Pages.

## Structure

- **Hero** — Name, tagline, primary CTAs
- **About** — Professional summary
- **Skills** — Languages, frontend, backend, databases, tools, concepts
- **Experience** — Internships (AI Research, HumbleWalking, Anand Housing)
- **Projects** — Institute Management System, Grocery Store Management System
- **Education** — B.E. EXTC, HSC, SSC
- **Leadership & Achievements** — Hackathons, full-stack projects, Agile
- **Contact** — Email, phone, LinkedIn, GitHub

## Tech stack

- React 18
- Vite 5
- CSS (custom properties, no framework)
